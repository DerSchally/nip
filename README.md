# NID Prototype - Österreichische Post

UI/UX showcase prototype for Österreichische Post AG's NID (Nachsendeauftrag Inland / Domestic Mail Forwarding) system.

## Quick Start

```bash
npm install
npm run dev
```

## Features

- 8-step mail forwarding form
- Austrian Post branding (yellow/black)
- PLZ → City auto-lookup (36 Austrian codes)
- 1-5 persons or company support
- Mock validation with loading states

## Tech Stack

- React 19 + Vite
- Tailwind CSS
- Lucide React icons
- date-fns
